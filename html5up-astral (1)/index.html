Goober Search

I’m proud of Goober, a hobby search engine I built with a team of four and now host on Vercel. We wrote an offline Python pipeline that crawls and fingerprint‑hashes content with SimHash to de‑duplicate 40 k+ public‑domain documents, then stores dense vector representations of documents in a index for sub 300  ms vector search.  Queries hit an API that fuses the vector scores with a ranker algorithm and returns results through a stylized React front end.  I learned a lot from my peers, who were all a year above me and taught me collaboration and development practices. Designing incremental reindexing and cache functions was my first taste of real‑world retrieval at scale, and the lessons, especially proximity vectorization, fed directly into the RAG pipeline I later shipped at Metric Analytics.
