Projects

Goober Search

Goober is a web search engine I collaborated on with 4 others. We got it to handle tens of thousands of pages while maintaining a query response time under 300ms! We tokenized both queries and documents with snowball stemming, and used an inverted index with tf-idf scoring to rank and display the top results. This was my first time working with a scaled dataset! To accommodate 40k+ files, we used pre processing and map reduce techniques to prevent overloading RAM.

Fablix Website

Fablix is one of my most ambitious web apps yet! I made it with one partner in crime, my roommate. It's a movie catalogue site with features like advanced browsing, cart checkout backed by sessions, full-text search with autocomplete, bot checks with reCAPTCHA, and more! It uses MySQL connection pooling. We also experimented with containerization by creating a docker file and running it in on Kubernetes clusters (1 admin and 2 worker t2.micros).
